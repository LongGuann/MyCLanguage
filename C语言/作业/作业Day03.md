  short int（二字节） 的取值范围是65535/2
-  以下选项中不正确的C语言字符型常量是（D）
  双引号引出的不是C语言字符型常量
- 在C语言中字符型数据在计算机中，以字符的（C）形式存储
 ASCII是字符型在计算机内存中的储存形式
- sizeof(double)的结果值是（A）
 double是双精度浮点型变量 在计算机中占有8个字节位

 \n属于char型变量 在计算机中占用1个字节位
  二、填空题  1   4    5   10     12
- C语言中的逻辑值真是用（1）来表示的，逻辑假使用（0）来表示
- 无符号基本整型的数据类型符为（unsigned int），双精度实型数据类型符为（double），字符型数据类型符为（char）
- C语言中有符号数和无符号数的本质区别是（最高位中有符号数据类型表示正负，无符号数据类型表示数据位）
- C语言中-15在计算机内存中8位二进制存放形式为（1000 1111）和15在计算机内存中存放方式一样的八位二进制数的十进制数为（143）
- 在C语言中，如下程序的输出结果为
```C
char c = 250;
unsigned char d;
char f;
d = c + 249;
f = c + 249;
printf("d = %d\n", d); // 243
printf("d = %u\n", d); // 243
printf("f = %d\n", f); // -13
printf("f = %u\n", f); // 4294967283
```
