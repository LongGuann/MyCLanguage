# 开发板网络配置

在开发板中`/etc/profile`文件中配置
```shell
vi /etc/profile

ifconfig eth0 up
# 这里给开发板设置静态ip
ifconfig eth0 192.168.8.3
route add default gw 192.168.8.253 dev eth0
echo "nameserver 202.96.128.86" > /etc/resolv.conf
echo "nameserver 202.96.128.166" >> /etc/resolv.conf
echo "search lan" >> /etc/resolv.conf

# 这个指令开机自动启动qt智能物联网应用,关闭它
# cd /IOT                                                   
# ./iot &                                                   
cd /
```
## 文件传输 nfs server

1. 先去配置nfs server共享文件夹
![](./nfs配置.png)

2. 去到开发板
    - 在/mnt中创建一个用于挂载远程nfs服务器的空目录 `mkdir /mnt/nfs`
    - 使用mount指令将远程服务器挂载至指定目录

```shell
mount -o nolock 192.168.8.你的ip:/share /mnt/nfs
```    

注意: 
1. 输入挂载指令时,一定不要先cd进nfs目录去,要在nfs目录外部输入
2. 记得关防火墙

## 网线连接开发板

### telnet协议连接开发板

1. 开发板设置telnet
打开开发板配置文件/etc/profile 添加telnet协议,保存退出

```shell
telnetd &
```

2. 设置密码
```shell
# 在终端中输入指令
passwd
```
一路回车，默认不需要密码
重启开发板

3. 使用SecureCRT连接
![](./telnet连接.png)
```shell
# 登录用户 root
GEC6818 login: root
# 密码回车
Password: 
```


### SSH协议连接开发板
1. 将压缩包SSH_config.tar.bz 下载到开发板根目录

2. 进入开发板根目录，解压缩，配置协议
```shell
cd  / 
tar   -xzvf   SSH_config.tar.bz 
cd   SSH/openssh/ 
./auto 
```
3. 配置ssh密码
```shell
passwd   root
``` 
4. 重启开发板 
```shell
reboot 
```
5. 使用SecureCRT 的 SSH2 协议进行连接
