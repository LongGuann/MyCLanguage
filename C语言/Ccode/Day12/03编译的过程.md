# 编译的过程

编译过程分为以下四个阶段：

1. **预处理**：处理预处理语句（#开头的语句），删除注释、头文件展开、宏替换...
```shell
gcc  hello.c  -o  hello.i  -E
```
2. **编译**: 将C语言程序转化为汇编语言 (机器语言)
```shell
gcc  hello.c  -o  hello.s  -S
```
3. **汇编**：将程序代码转化为二进制代码
```shell
gcc  hello.c  -o  hello.o   -c
```
4. **链接**：将所有二进制代码合并起来，根据应用规则生成一个专门针对某个平台执行的应用程序镜像
```shell
 gcc  hello.c  -o  hello
```
